1. Extract all the files from the zip file
2. Load the HW4.java into eclipse by adding the required Lucene jars
3. Make sure you have the raw files for all the documents that needs to be indexed in a folder and give the locatin of that folder when prompted
4. Run the application and give the queries to get the documents and score for each query 
5. Go to the directory where you have BM25.py in your terminal/command prompt
6. To run the scoring program, give the command "python BM25.py". When prompted for query file path, type in the full path where the file is located
7. Make sure you have the vocabulary.txt and uniIndex.txt in the same folder as the python file before executing the code
8. Score files in the required output will be created for each query in the same folder
