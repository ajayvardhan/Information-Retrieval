1. Download parser.py from the zip file
2. Go to the directory where you have the file in your terminal/command prompt
3. To run the pagerank program, give the command "python pagerank.py". When prompted for input files path, type in the full path where the files are located. When prompted for the output files path, give the path where the corpus files needs to be generated
4. All the index and output files will be generated in the same directory where the source code is, after the execution
