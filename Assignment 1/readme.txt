1. Download crawler.py and focusedCrawler.py from the zip file
2. Go to the directory where you have both these files in your terminal/command prompt
3a. To run the Crawler program, give the command "python crawler.py". When prompted for seed url, give a valid complete wikipedia URL (eg: "https://en.wikipedia.org/wiki/Tropical_cyclone").
3b. To run the Focused Crawler program, give the command "python focusedCrawler.py". When prompted for seed url, give a valid complete wikipedia URL (eg: "https://en.wikipedia.org/wiki/Tropical_cyclone") and a keyword which you want to focus the search on.
4. The list of crawled URLs will be added to a file called Output.txt in the same directory of the crawler files.

Maximum depth:

For crawler - 5
For Focused Crawler - 6
