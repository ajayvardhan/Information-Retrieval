1. Download pagerank.py from the zip file
2. Go to the directory where you have the file in your terminal/command prompt
3. To run the pagerank program, give the command "python pagerank.py". When prompted for file name, give a file name that contains the inlinks graph. If the file is in the same directory as the code, just give the file name (Eg: "test.txt")
4. List of pages and the corresponding pageranks will be available in output.txt in the same directory
